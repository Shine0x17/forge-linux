# OverTheWire Bandit

This directory contains my learning records for the OverTheWire Bandit wargame.
Each level is documented using a consistent structure to track goals, actions,
commands, reading notes, and personal understanding.

---

## Level File Structure

### Goal
The objective of the level.

### What I did
Actions taken to complete the level.

### Command
Commands used during the level.

### Reading Notes
Notes summarized from the provided or referenced reading materials.

### Mastery Check
A self-check to confirm understanding of the level.
